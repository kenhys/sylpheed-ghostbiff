
メールチェック プラグイン
================================================================================

このプラグインはメール受信時にDirectSSTPにより新着通知を行うプラグインです。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3

SSP 2.2.48f
http://ssp.shillest.net/

インストール方法
--------------------------------------------------------------------------------

ghostbiff.dllをSylpheedのプラグインディレクトリへとコピーします。

1) Sylpheedのzipアーカイブを使用している場合

pluginsディレクトリへ配置します。

2) Sylpheedをインスーラによりセットアップした場合

%APPDATA%/Sylpheed/plugins以下に配置します。
Windows XPの場合以下の場所にdllをコピーして下さい。
C:\Documents and Settings\(ユーザアカウント)\Application Data\Sylpheed\plugins

1)もしくは2)によりセットアップは完了です。
問題なければ ステータスバーに ベルアイコン が表示されるはずです。

使い方
--------------------------------------------------------------------------------

SSPをあらかじめ起動します。
SSPが起動していない場合、ghostbiffによる新着通知は行われません。

ステータスバーのベルアイコンをクリックしてプラグインを有効にします。

メールを受信するたびにDirectSSTPによりバルーンに新着通知を表示します。

注意事項
--------------------------------------------------------------------------------

本プラグインには評価実験用としてアクエスト社のAquesTalk2およびAqKanji2Koe
(いずれも評価版)に対応しています。

その場合、sylpheed.exeと同じところにAqKanji2Koe.dllおよびAquesTalk2Da.dllが必要となります。
