
メールチェック プラグイン
================================================================================

このプラグインはメール受信時にポップアップにより通知するだけのプラグインです。

動作環境
--------------------------------------------------------------------------------

Windows XP SP3


インストール方法
--------------------------------------------------------------------------------

1. dllを配置

ghostbiff.dllをSylpheedのプラグインディレクトリへとコピーします。
Sylpheedのzipアーカイブならpluginsディレクトリへ配置するだけです。

もしくは%APPDATA%/Sylpheed/plugins以下に配置します。

これでセットアップは完了です。
問題なければ () が表示されるはずです。

使いかた
--------------------------------------------------------------------------------

本プラグインは常時有効なため特に操作は必要ありません。
